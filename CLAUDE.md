## Agent skills

### Issue tracker

Issues are tracked on GitHub Issues for the `Jakeming/AI-novel-workshop` repository. See `docs/agents/issue-tracker.md`.

### Triage labels

Uses the default five-state label vocabulary (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

Multi-context layout. A `CONTEXT-MAP.md` at the repo root points to per-context `CONTEXT.md` files. See `docs/agents/domain.md`.
